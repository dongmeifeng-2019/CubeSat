This folder contains the scripts used for recreating Figures 4-9 in the paper.
If you would like to reproduce these figures, please: 
1, download this repo CubeSat to your local machine; you need to download everything in this repo
2, open the script, and change line 4 to setwd('YOUR WORKING DIRECTORY')
   'YOUR WORKING DIRECTORY' refers to the location of the folder CubeSat
